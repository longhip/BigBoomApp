## Steedoffice Core (MVC).

Author: longhip.dev@gmail.com

## Clone

```sh
$ git clone https://github.com/longhip/steedoffice_core_module.git
```
## Install package for app

```sh
$ cd /path/to/project
```
```sh
$ npm install
```

## Composer update dependencies for api

```sh
$ cd /path/to/project
```
```sh
$ cd api
```
```sh
$ composer update
```

## Connect mysql
You can connect with .env file

## Migrate and Seeder

```sh
$ cd /path/to/project
```
```sh
$ cd api
```
```sh
$ php artisan migrate
```
```sh
$ php artisan db:seed
```
